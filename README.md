#  Churn Modelling – Task 3


- Perform **Exploratory Data Analysis (EDA)** on customer data  
- Identify key factors influencing customer churn  
- Build and evaluate machine learning models  
- Predict whether a customer will leave the bank

---

##  Tools & Technologies
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-learn  

---

##  Dataset Features
- Credit Score  
- Geography  
- Gender  
- Age  
- Tenure  
- Balance  
- Number of Products  
- Has Credit Card  
- Is Active Member  
- Estimated Salary  

---

##  Project Workflow
1. **Data Preprocessing**  
   - Handling missing values  
   - Encoding categorical variables  

2. **Exploratory Data Analysis (EDA)**  
   - Data visualization (histograms, boxplots, heatmaps)  
   - Correlation analysis  

3. **Model Building**  
   - Logistic Regression  
   - Decision Tree  
   - Random Forest  

4. **Model Evaluation**  
   - Accuracy score  
   - Confusion matrix  
   - Precision, Recall, F1-score  

---

##  Expected Results
- Identify customers at high risk of churn  
- Provide actionable insights for customer retention  
- Predict churn using machine learning models
